  - T006 (`proc.go`) blocks T007, T016, T017, T018, T019.

  - T022 (`path.go` + shared resolver) gates T023, T024, T026 because they reuse the resolver helper.
  - T025 (`cd.go`), T027 (`ls.go`), T028 (`shell_init.go`), T029 (`config.go`) are independent of T022 and can run in parallel.
- Phase 4 (T030–T032) starts after Phase 3.
- Phase 5 (T033–T034) starts after Phase 4.
- Phase 6 (T035–T043) is the gate; runs strictly sequentially in numbered order.
